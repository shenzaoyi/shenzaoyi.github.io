    1. docker依赖于宿主内核，提供自己的根文件系统，对于宿主机来说就是一个根文件隔离的进程(chroot)，Docker镜像实际上就是一个分层文件系统(见REFERENCES).

    Image #

    Docker image 相当于linux中的root文件系统;使用Union FS减小体积，层层构建

    Container #

    Container 实质上是进程，一种运行于自己独立的运行空间的进程
