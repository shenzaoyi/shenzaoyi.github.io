SVG Editor

SVG

可伸缩矢量图(Scalable Vector Graphics)，基于 XML 描述二维矢量图。SVG 格式提供的是矢量图，这意味着它的图像能够被无限放大而不失真或降低质量，并且可以方便地修改内容，无需图形编辑器。

width/height: 宽度，高度； xmlns: 指定命名空间， 使用标签来描述不同图形。
