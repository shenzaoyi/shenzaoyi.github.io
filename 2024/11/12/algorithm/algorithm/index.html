字符串的前后缀定义

前缀函数

定义

给定长度为 n 的字符串， 其前缀函数定义为长度为 n 的数组 $\pi$, $\pi[i]$ 定义如下：

  1. $\pi[0]$ = 0；
  2. i > 0, $\pi[i]$ ，字串 s[0,i]相等真前缀和真后缀中，最长的真前缀真后缀的长度, 如果没有，则等于0。

即： $\pi[i] = max_{k=0…i} { k: s[0,k-1] = s[i-k+1,i] }$
